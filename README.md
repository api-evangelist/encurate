# Encurate (encurate)

Encurate provides a Keto Diet API offering over 450 keto recipes across 11 categories (drinks, smoothies, staples, dips, desserts, soups, fish, beef, appetizers, snacks, and breakfast) with structured nutritional values including calories, fats, carbohydrates, and proteins. The REST API returns JSON and is distributed via RapidAPI with a free testing tier and a paid unlimited production tier for integration into health, fitness, and nutrition applications.

**URL:** [Visit APIs.json URL](https://encurate.app/keto_diet_api/)
